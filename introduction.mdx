## Welcome to EasyACT

EasyACT is a personal finance management web application designed to make expense tracking simple and insightful. With an intuitive interface and powerful visualization tools, you can easily record daily transactions, analyze your spending patterns, and take control of your financial future.

## Technology stack

EasyACT is built with modern, production-ready technologies:

### Frontend

- **Vue 3** (3.5.25) - Progressive JavaScript framework
- **PrimeVue** (4.5.3) - Rich UI component library
- **Tailwind CSS** (4.1.18) - Utility-first CSS framework
- **Chart.js** (4.5.1) - Beautiful data visualization
- **Pinia** (3.0.4) - Intuitive state management
- **Vite** (7.2.4) - Lightning-fast build tool

### Backend

- **Spring Boot** (4.0.1) - Enterprise Java framework
- **Java** (25) - Robust programming language
- **Spring Data JPA** - Simplified data access
- **PostgreSQL** (15) - Reliable relational database
- **SpringDoc OpenAPI** (2.6.0) - Automated API documentation

### Deployment

- **Docker Compose** - One-command deployment
- **Nginx** - High-performance web server
- **PostgreSQL** - Persistent data storage

## System architecture

EasyACT uses a modern three-tier architecture with containerized deployment:

```
┌─────────────┐      ┌─────────────┐      ┌─────────────┐
│             │      │             │      │             │
│   Web UI    │─────▶│   API       │─────▶│  Database   │
│  (Vue.js)   │      │ (Spring     │      │ (PostgreSQL)│
│   Nginx     │      │  Boot)      │      │             │
│             │      │             │      │             │
└─────────────┘      └─────────────┘      └─────────────┘
   Port: 8000           Port: 8080          Port: 5432
```

<Note>
  All three services run in Docker containers and communicate through a private bridge network for security and isolation.
</Note>
